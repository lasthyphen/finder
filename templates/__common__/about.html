About Dijets Search Engine

Dijets Search is a fully decentralised metasearch engine that aggregates results from several search engines, and sources to output responses based on the combined score for each of the suggested links. Dijets Search is completely ad-free with absolutely no trackers that could passively observe your search queries.

With Dijets Browser

You can download Dijets Browser to take full advantage of Dijets Decentralised Apps and Services. Dijets Search Engine is the default search provider within the browser and you get tonnes of cool features including a digital wallet and all of Dijets services already integrated as extensions.

Why use Dijets Search?

  • Dijets may not offer you as personalised results as Google, but it doesn't generate a profile about you either.
  • Put simply, Dijets search engine doesn't care about what you search for, never shares anything with a third party, and it can't be used to compromise you.
  • Dijets Search is completely free to use and is part of Dijets growing ecosystem of Decentralised products and services. Read more about Dijets Ecosystem.

If you do care about privacy, want to be a conscious user, or otherwise believe in digital freedom, make Dijets your default search engine. You can even set Dijets as a default search service provider within Dijets Browser.

Technical details - How does it work?

Dijets Search is a metasearch engine, with enhanced features that provide you with unparalleled security and anonymity. There are DLT-Specific features that aren't available on any other search engines. Furthermore, Dijets Search Engine is packed with general utility features like:

  • Export your Search Results in a CSV, JSON or RSS format.
  • Play Media files within the same window, without having to manage multiple tabs.
  • Navigate Maps and search locations from the same query results page.
  • Modular searches based on individual preference. Choose to include or exclude engines you don't want to query.
  • Route your search query through dual proxies to completely mask your IP from the serving pages.
Search queries submitted on Dijets are mixed up with searches on other platforms without storing search data. Queries are made using a POST request on every browser (except Chromium-based browsers*). Therefore they can neither show up in our logs, nor your url history. In the case of Chromium-based browser users there is an exception: searx uses the search bar to perform GET requests. Dijets Search can be added to your browser's search bar; moreover, it can be set as the default search engine.

How to set as the default search engine?

Dijets Search supports OpenSearch. For more information on changing your default search engine, see your browser's documentation:

  • Microsoft Edge
  • Chromium-based browsers only add websites that the user navigates to without a path.
